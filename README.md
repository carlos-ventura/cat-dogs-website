**Run**
    Navigate into your new site’s directory and start it up.

    cd cat-dogs-website/
    npm run develop

**Open website**

Site is now running at http://localhost:8000

**Start server**

    cd cat-dogs-website/server
    ts-node server.ts

**Open server website**

Server is now running at http://localhost:9000
